# war 파일 실행

```bash
cp mc.war /etc/apache-tomcat-9.0.22/webapps/
cd /etc/apache-tomcat-9.0.22/webapps/
[root@server3 webapps]# ls
ROOT  docs  examples  host-manager  manager  mc.war
starttomcat
[root@server3 webapps]# ls
ROOT  docs  examples  host-manager  manager  mc  mc.war
mc 파일이 자동으로 생기는 것 확인
```

브라우저에서 아이피주소로 접속 확인
